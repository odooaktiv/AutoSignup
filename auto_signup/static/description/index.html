Auto Sign Up

The Odoo’s default feature allows user to switch to auto sign up by configuring the portal access for- “Allow external users to sign up” from the settings. This module allows user to get the Sign Up button automatically on the screen when user installs this module without any configurations.

Prerequisites before Installation

No prerequisites needed.

Configurations

No configurations.

Earlier if user tried to get an automatic sign up button then they have to configure it through following steps: Go to General Settings> Under Portal Access> Make the “Allow external user sign up” Boolean as true.

But if user install this module, then they will not have to configure the process from backend. Simply install this module and sign up button will show up besides the log in button.
